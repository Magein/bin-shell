### 脚本说明
    
    安装docker  使用 ./start.sh --help 查看帮助
     
    首先要检查安装环境，安装步骤使用官网给出的步骤，如果非root用户，需要具备sudo权限
       
    1. 要具备使用 yum 的权限
     
    2. 要具备使用 yum-config-manager 的权限
    
    options:
     
        -r,--remove  移除已经安装的docker （需要具有上述sudo权限）
        
        -c,--check   检测是否具备安装条件
        
        -s,--show    查看可选择的安装版本  (用到yum-config-manager)
